Angular-meals

Mock for an app. The app takes pictures to your meals and see the percentages of calories, proteins, etc. in seconds.

Resources
  • Material AngularJS
  • Bootstrap
  • JQuery

Testing WebApp with AngularJs and Material Design.

Steps to get the source code

  • git clone git@github.com:foodSquare/angular-meals.git
  • cd angular-meals
  • npm install
  • bower install
  • grunt build
  • localhost:8000
